# Periodic Table

A personal project that implements a **web-based Periodic Table application**.  
The project is built using **Embedded JavaScript (EJS)**, **SASS**, and **JavaScript**, bundled with **Webpack**.

## 📌 Features

The Periodic Table displays information about chemical elements, including:

- Name
- Symbol
- Atomic number
- Atomic mass
- Density
- Melting point
- Boiling point

This type of visualization is widely used in **chemistry, physics, and other scientific fields** as a reference for understanding element properties and relationships.

## 🚀 Getting Started

Follow these steps to run the project locally.

1. Clone the repository

```bash
git clone https://github.com/your-username/periodic-table.git
cd periodic-table
```

2. Install dependencies

```bash
yarn install
```

3. Start development server

```bash
yarn dev
```

This will start Webpack Dev Server, compile the project, and launch a local development server with hot reload enabled.

After running the command, open the URL shown in the terminal (typically):

```text
http://localhost:8080
```
